# diy-chat-go-react

## how to run

```
# git clone https://github.com/chelseafandev/diy-chat-go-react.git
```

### backend
```bash
# cd diy-chat-go-react
# cd backend
# go run main.go
Distributed Chat App v0.01
```

### frontend
```
# cd diy-chat-go-react
# cd frontend_insta
# sudo npm i react-scripts
# npm start
```
